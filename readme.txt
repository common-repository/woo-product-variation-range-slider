Woo Product Variation Range Slider

== Description ==
Customer can choose variable product multiple variations with the help range slider.

== Installation ==
- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select ‘Add New’ from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress

Or 

- Upload the woo-product-variation-range-slider (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress
